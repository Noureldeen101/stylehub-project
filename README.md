# StyleHub Project

StyleHub is a frontend clothing e-commerce website project built using HTML, CSS, and JavaScript.

## Features
- Home page
- Men and Women product pages
- Product detail pages
- Cart page
- Checkout page
- Visa and cash on delivery payment options

## Technologies Used
- HTML
- CSS
- JavaScript

## Pages
- home.html
- men.html
- women.html
- cart.html
- checkout.html

## How to Run
Download the project files and open `home.html` in the browser.
